# ED Exobiology

Crate for predicting exobiology species for a planet based on provided information. This is currently work in
progress as I'm still working out the conditions for each species. Builds on top of
the [ed-journals](https://crates.io/crates/ed-journals) crate.

## Features and plans

- [x] Spawn conditions and resolution.
- [x] Structs to easily fill in with information to perform predictions.
- [ ] Improvements are still being made to prediction conditions.

# Contributing

If you think that species are not predicted correctly, or that predictions needs to be tweaked: don't hesitate to
[open an issue](https://github.com/rster2002/ed-journals/issues/new). Getting this right can be hard so any input,
even a single test case, helps make the predictions better in the future.
